INFORMÁTICA PARA NIÑOS

La informatica.- se refiere al procesamiento (acción de leer algo) automático de información mediante dispositivos electrónicos y sistemas computacionales (componentes físicos que interactúan mediante instrucciones dadas).

La computadora.- es un dispositivo electrónico que procesa automáticamente grandes cantidades de datos de manera rápida y sin errores.

La computación.- no es lo mismo computación que computadora la computación se usaba como un método universal para resolver problemas matemáticos.

Programación.- es un proceso por el cual una persona desarrolla un programa usando una herramienta para poder escribir el código, el cual está escrito en lenguaje de programación.

Lenguaje de programación.- es un programa que construye otros programas su nombre se debe a que comprende n lenguaje diseñado para organizar algoritmos (secuencia de instrucciones que representa una solución a un problema determinado) y procesos lógicos.

Características de un programa

Eficacia.- Un programa debe resolver el problema para el cual fue diseñado.

Claridad.- un programa debe ser legible.

Eficiencia.- el programa usa de manera óptima los recursos.

Portabilidad.- el programa tiene la capacidad de poder ejecutarse en una plataforma diferente a la que se usó para desarrollarla o programarla.

dato.- es la unidad máxima de información que por sí solo no tiene sentido alguno.

    Software de base: Sistema operativo (SO)

    El Sistema Operativo en una computadora tiene el objeto de interpretar las órdenes del usuario y comunicárselas a la computadora, es decir, sirve para que el usuario la computadora puedan interactuar.

    Tipos de SO

    Existen SO: MONOUSUARIOS y MULTIUSUARIOS.

    Monousuario: Los SO monousuarios son aquellos que nada más puede atender a un solo usuario: DOS, CP/M.

    Multiusuario: Permite que dos o más usuarios utilicen sus programas al mismo tiempo, en lo que se denominan redes de computadoras. Por ejemplo: el SO UNIX o el WINDOWS.

    SO actuales

    Linux: SO de distribución libre (gratuita) (distribución de Linux).

    Mac OS: SO de las computadoras la empresa Apple.

    Windows: S0 de las computadoras provee licencias con costo.

    Android: SO de los celulares, especial para dispositivos móviles con pantalla táctil (touch-screen).

    Software de base: Lenguajes de programación.

    Es un lenguaje artificial, posee reglas de escritura bien definidas (sintaxis), un LP le proporciona a una persona (desarrollado con la capacidad de escribir una serie de instrucciones o secuencias de órdenes con el fin de controlar el comportamiento físico o lógico de una Computadora.

    Tipos de lenguaje de programación

    Bajo Nivel: Su sintaxis se basa en el Lenguaje de Máquina, desde Os y 1s, hasta lenguajes de tipo Ensamblador (que usan mnemónicos (lenguaje de máquina))

    Alto Nivel: Su sintaxis se basa en el Lenguaje Humano, es decir que las instrucciones que el programador escribe en un programa se parecen mucho a las expresiones verbales del ser humano (INGLÉS).

    Nivel Medio: Comparten la sintaxis de ambos "mundos".

    Es software horizontal, creados para tareas generales, como ejemplo la Suite Office.

    Áreas relacionadas con la informática

    Análisis de Sistemas. - Integra técnicas para realizar la recopilación de información, donde se estudian procedimientos para planear y diseñar sistemas, además de analizar los sistemas existentes para mejorar su operación o al menos evitar su degradación.

    Ingeniería de Sistemas. - Es la aplicación de los conocimientos científicos para el análisis, diseño e implementación de sistemas de información, complejos, haciendo uso de técnicas de modelaje, simulación y la filosofía del enfoque de sistemas.

    Cibernética. – Termino introducido por el matemático Norbert Wiener en 1948. Ciencia interdisciplinaria que estudia las leyes generales de la transformación de la información y los sistemas de control. Su estudio abarca tanto a máquinas como a seres vivos.

    Robótica. - Palabra que proviene de “robota”, que significa “trabajo obligatorio”.

    La palabra robot se usaba para referirse a una máquina que realiza trabajos para ayudar a las personas o que efectuaba tareas difíciles o desagradables para los humanos.

    Los robots tienen la capacidad de desarrollar actividades que caracterizan a la inteligencia humana.

    Sistemas Expertos. - Se basa en emular la respuesta de un experto en algún área, ante una situación de la vida real. Son una de las aplicaciones de la inteligencia artificial que pretende simular el razonamiento humano.

    Inteligencia Artificial. - Estudia la posibilidad de lograr que las máquinas (computadoras) realicen tareas que, por el momento son realizadas mejor por los seres humanos. Combinación de algoritmos planteados con el propósito de crear máquinas que presentan las mismas capacidades del ser humano.

    Telemática. - Conjunción de las palabras telecomunicaciones informática. Estudia el conjunto de técnicas que se usan para transmitir datos de un sistema informático a otro.

    Con un mundo cada vez más conectado el uso de la telemática no tiene fin ya que se amplía cada vez más.

    REDES

    Una red de computadoras son varias computadoras conectadas entre si que pueden comunicarse y comparten recursos.

    Clasificación de redes

    Local Area Network - Red de área local (LAN): Limitado físicamente a un edificio o un entorno a pocos kilómetros.

    Metropolitan Area Network – Red de área metropolitana (MAN): Red de alta velocidad que brinda cobertura en un área geográfica extensa.

    Wide Area Network – Red de área amplia (WAN): Se extiende en un país o continente (INTERNET)

    Storage Area Network – Red de área de almacenamiento (SAN o NAS): Es una red de almacenamiento integral (actividades enfocadas en el mejor almacenamiento y cuidado de la mercancía).

    Componentes de una red

    Hub (Concentrador): Permite conectar entre sí otros equipos y retransmite la información que recibe desde cualquiera de ellos a todos los demás.

    Router (Enrutador): Interconecta trozos o redes enteras. Toma decisiones lógicas con respecto a la mejor ruta para enviar de datos a través de una red.

    Switch (Conmutador): Entrega datos de acuerdo a la dirección de destino.

    Hosts (Huésped): Son equipos computacionales que inician y procesan la información proveniente de sus pares.

    TIPOS DE TOPOLOGÍAS

    Topología BUS: Todos sus nodos están conectados directamente a un enlace y no presenta conexión entre nodos una ruptura del cable que conecta los hosts haría que la red se desconecte.

    Topología en estrella: Los hosts están conectados directamente al servidor y todas las comunicaciones se realizarán necesariamente a través del equipo central.

    Topología en anillo: Cada hosts tiene un receptor y un transmisor que pasa la señal al siguiente hosts del anillo, si algún nodo de la red se cae la comunicación de todo el anillo se perderá.

    Conceptos para realizar algoritmos.

    Constante: Es un objeto de valor invariable durante el proceso que compone un algoritmo.

    Variable: Es un objeto cuyo valor puede variar cuando se ejecutan los pasos de un algoritmo.

    Características de una variable

    Nombre: También llamado identificador en los lenguajes de programación.

    Tipo de dato: Indica que tipo de valores tendrá una variable.

  • Numéricos (enteros o decimales).
  • Alfabéticos (secuencia de letras o números).
  • Booleanos (solo acepta valores de verdad o falso).

    Tamaño: Cada tipo de dato tiene un tamaño en bytes.

    Contenido: Este atributo es opcional, la variable puede contener un valor definido desde antes de empezar un algoritmo.
